Below is a description of each field in the Portrait Dataset csv file:

index - The row index from 0 to 6885 of each portrait, the order the images show up in the .zip folder

filename - The file name of each portrait as it appears in the .zip folder

title - The title of the portrait from artstor

full_year - The year listed on artstor with moderate cleaning to extract the year from the full date, note some inconsistent formatting and date ranges for some portraits

start_year - The earliest possible year for portraits with a date range on artstor

mid_year - The midpoint year for portraits with a date range on artstor

end_year - The latest possible year for portraits with a date range on artstor

circa - 'yes' denotes portraits that have a circa indicator in the full_date field (ex. "c.", "ca.", "circa")

continent - The continent where the portrait originated

country - The country where the portrait originated

artstor_url - The link to the artstor listing

image_url - The link to the image thumbnail

artstor_artist - The artist from the artstor listing with moderate cleaning

first_artist - The first or only artist of the portrait

second_artist - If applicable, the second artist of the portrait

notes - If applicable, non-essential notes about the artist(s)

attributed_to - If applicable, the artist the portrait is attributed to

attributed_to_2 - If applicable, the second artist the portrait is attributed to

attributed_to_3 - If applicable, the third artist the portrait is attributed to

copy_of - If applicable, the original artist that the current portrait is a copy of

in_style_of - If applicable, the original artist that the current portrait is in the style of

workshop_of - If applicable, the artist of the workshop where this portrait was created

circle_of - If applicable, the circle of the artist where this portrait was created
